# scraping-app（料金スクレイピングWebアプリ）

Phase 2「スクレイピング」で作る、URLを入力すると料金情報を取得して、テーブル・比較表・CSVで出力する Webアプリです。
Playwright でブラウザを自動操作し、ペアのデモサイト `scraping-target` の料金ページから料金プランを取得します。

## できること

- URL入力フォーム（最初からデモサイト5社分のURLが入っています）
- 「実行」ボタンで順番にスクレイピング（**3秒間隔**でアクセス＝マナー）
- プログレスバーで進捗表示
- 取得結果をテーブル表示
- 5社の月額を並べた比較表（最安をハイライト）
- CSVダウンロード

## 使い方（はじめての人向け）

このアプリは、ペアのデモサイト `scraping-target` を**同時に起動**して使います。

### 1. デモサイトを起動する（ターミナル①）

`apps/scraping-target` フォルダで：

```bash
npm install
npm run dev
```

→ デモサイトが `http://localhost:3000` で立ち上がります。このターミナルはそのままにしておきます。

### 2. このアプリを起動する（ターミナル②）

`apps/scraping-app` フォルダで：

```bash
npm install
npm run dev
```

> `npm install` のときに、Playwright が使うブラウザ（Chromium）も自動でダウンロードされます（`postinstall`）。
> もし自動で入らなかった場合は、手動で次を実行してください。
>
> ```bash
> npx playwright install chromium
> ```

→ このアプリが `http://localhost:3100` で立ち上がります（デモサイトと port がぶつからないように 3100 にしています）。

### 3. ブラウザで開いて実行する

[http://localhost:3100](http://localhost:3100) を開き、「実行」ボタンを押します。
5社分の料金が順番に取得され、テーブル・比較表・CSVが使えるようになります。

## 仕組み（フォルダの見どころ）

- `app/page.tsx` … 画面側。URLを1件ずつ順番にAPIへ送り、3秒間隔を空けながら進捗を表示します。
- `app/api/scrape/route.ts` … サーバー側。Playwright でページを開き、料金プランを取り出します。
- `lib/types.ts` … 取得結果の型定義。

## 注意

- このアプリは学習用のデモです。スクレイピングは、対象サイトの利用規約・robots.txt・アクセス間隔を必ず守って利用してください。
- デモサイト以外（実在のサイト）のHTML構造は異なるため、そのままでは取得できません。対象サイトに合わせてセレクタ（`.company-name` など）の調整が必要です。
